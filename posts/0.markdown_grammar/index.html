标题的使用方式 (Title or Heading)#

1~6级标题分别用: 1~6个# 空格 来表示

  • 代码
# heading 1
## heading 2
### heading 3
#### heading 4
##### heading 5
###### heading 6
  • 效果

heading 1#

heading 2#

heading 3#

heading 4#

heading 5#
heading 6#

换行以及分段 (Rows and Paragraphs)#

  • 换行: 空格 空格 换行

    • 代码
    row1  <!--Here are two spacebar-->
row2
    • 效果

    row1
    row2

  • 分段: 换行 换行用来表示

    • 代码
    paragraph1

paragraph2
    • 效果

    paragraph1

    paragraph2

设置字体属性 (Character Attribute)#

~~包住表示删除线
1~3个*包住分别表示斜体、粗体、粗斜体

  • 代码
~~strikeout~~ ~~删除线~~  
*italic* *斜体*  
**bold** **粗体**  
***italic & bold*** ***粗斜体***
  • 效果

strikeout 删除线
italic 斜体
bold 粗体
italic & bold 粗斜体

列表与序号 (List and Order)#

  • 无序列表: -空格或 +空格或*空格, 后面可接[空格]空格或者[x]空格来表示复选框
    为了兼容性, 不建议混合使用, 尽管下例如此, 但只是为了演示

    • 代码
    - list 1
+ list 2
* list 3
- [ ] 这个列项没有选中  
- [x] 这个列项被选中了
    • 效果
    • list 1
    • list 2
    • list 3
    • 这个列项没有选中
    • 这个列项被选中了

  • 有序列表: N.空格, 其中N表示起始序号
    后续数字大小无效,列表将从N开始递增, 详见下方例子

    • 代码
    2. order 2
1. order 3
5. order 4
    • 效果
    1. order 2
    2. order 3
    3. order 4

  • 子列表: 空格 空格 空格 空格 或 TAB

    • 代码
    - list
- list
    - child list
    - child list
    - child list
        - c c list
        - c c list
1. order
    1. child order
    2. child order
        1. c c order
        2. c c order
    • 效果
    • list
    • list
      • child list
      • child list
      • child list
        • c c list
        • c c list
    1. order
      1. child order
      2. child order
        1. c c order
        2. c c order

代码与代码块 (Code and Block)#

  • 突出强调(代码): 首尾用`包住

    • 代码
    This is the `keyword`.
    • 效果

    This is the keyword.

    代码中包含`时, 用``与空格包住, 如是: ``空格代码空格``

    • 代码
    `` Press the `keyword`. ``
    • 效果

    Press the `keyword`.

    同理若代码中含有n个`,则需要在代码首尾用n+个反引号与1个空格包住即可
    如下方举例, 代码原文中有两个反引号, 则代码本身需要在首尾用三个或以上(例子中使用了5个)反引号与一个空格来包住

    • 代码
    ````` We have two symbols: ``. `````
    • 效果

    We have two symbols: ``.

  • 代码块: 首尾各用 ``` 包住
    可在首3个`后面指定代码语言 , 如是: ```language代码块```
    除了反引号也可使用波浪号: ~~~, 详见下方举例

    • 代码
    ```  
#include <stdio.h>  //hh  
#define pi 3.1415926  
```  

    ```C  
#include <stdio.h>  
```  

    ~~~c++  
int main()  
{  
  return 0;  
}  
~~~
    • 效果
    #include <stdio.h>  //hh  
#define pi 3.1415926  

    #include <stdio.h>   

    int main()  
{  
  return 0;  
}

    若代码原文中有3个`或~, 则可用3+个`或~来包住代码块

    • 代码
    `````C
char* str=" ``` ";
printf("%s",str);
`````
    • 效果
    char* str=" ``` ";
printf("%s",str);

    除使用反引号与波浪号外, 还可以让整个代码块缩进4个或以上空格

    • 代码
        int a=0, b=0;
    int c=a+b;
    printf("c: %d", c);
    • 效果
    int a=0, b=0;
int c=a+b;
printf("c: %d", c);

引用 (Quotes)#

每行行首添加 >空格, 多层引用则增加>数, 见下例
为了兼容性, 建议在引用前后各增加一行空白行

  • 代码
content...

> quote 1
> quote 2
>
> another quote 1
> another quote 2
>> child quote 1
>>> c c quote 1

content...
  • 效果 (请忽略第一个引用, 此为本文引用)

content…

quote 1 quote 2

another quote 1 another quote 2

child quote 1

c c quote 1

content…

为了兼容性, 建议在前后添加空白行

  • 代码
content1...  
  
---  
  
content2...
  • 效果

content1…

content2…

超链接 (Links)#

  • 标准显示方式: []包住链接文本, 其后()包住链接地址, 地址后方添加空格然后用""包住可选提示
    如是: [显示文本](链接地址 "可选提示")

    • 代码
    [我是一个链接](www.xxxxxx.com)  
[谷歌](www.google.com "当你鼠标指向这个链接时, 我会显示这条提示")  
[AKIBA BLOG](syakiba.github.io "前往AKIBA的个人博客")
    • 效果

    我是一个链接
    谷歌
    AKIBA BLOG

  • 直接显示链接地址时,可以使用<>包住

    • 代码
    <www.google.com> *注意此方式需要http前缀,请见下方效果*  
<https://www.google.com>  
<xxxxx@gmail.com>
    • 效果

    <www.google.com> 注意此方式需要http前缀,请见下方效果
    https://www.google.com
    xxxxx@gmail.com

  • 引用链接分两个部分, 引用标识和引用内容
    引用标识: [显示文本] 空格 [引用标识]
    引用内容: [引用标识]:空格 链接地址 空格"可选提示" (可选提示可以用单引号、双引号、括号包住; 链接地址可以用尖括号包住也可以直接写)
    不建议使用, 作者编写本文时, 此方式一直失败

    • 代码
    [谷歌搜索] [0]  
[AKIBA BLOG] [a]
(other content)  
[0]: <https://www.google.com>  
[a]: syakiba.github.io "前往AKIBA的个人博客"
    • 效果

    [谷歌搜索] [0]
    [AKIBA BLOG] [a]
    (other content…)
    [0]: https://www.google.com
    [a]: syakiba.github.io “前往AKIBA的个人博客”

|用来表示表格列数, 第二行须要3个或以上-来标识表格且列数要与表头一样
-的左右可以加上:用来表示左右对齐(或居中), 如是: :- :-: -:

脚注#

两个部分组成, 脚注标识与脚注内容, 脚注内容为当前脚注标识与下一个脚注标识之前的文本
脚注标识: [^标识]
脚注内容: [^标识]: 空格 脚注内容
不建议使用, 为了兼容性建议脚注内容后面分段

  • 代码
脚注标识[^1]可以是字母或数字, 使用脚注[^2]功能前需确认当前处理器是否支持, 因为不是所有的markdown处理器[^3]都支持此功能.
